# Commercial Closer

Build agreement principles and pricing strategy for deals entering negotiation.

## When to Use
- Deal moving from Validating to Negotiating
- User asks about pricing, commercial terms, or agreement structure
- Preparing a proposal or SOW

## Process

### Step 1: Assess Commercial Readiness
Before building pricing, verify:
- [ ] Pain quantified in prospect's own words
- [ ] Economic Buyer identified and engaged
- [ ] Decision criteria confirmed
- [ ] Competition understood
- [ ] Champion can articulate ROI internally

If any are missing, flag them. Don't build pricing on an unqualified deal.

### Step 2: Build Value Bridge
Map prospect's stated pain to quantified business impact:

| Pain Point | Current Cost | With {{company.name}} | Delta |
|---|---|---|---|
| [Pain 1] | $[X]/year | $[Y]/year | $[savings] |
| [Pain 2] | [hours/week] | [hours/week] | [time saved] |

The value bridge must use THEIR numbers, not yours.

### Step 3: Structure the Offer
Design pricing options (recommend 3-option framework):

**Option A (Baseline)**: Minimum viable scope. Solves the stated pain.
**Option B (Recommended)**: Full scope. Addresses pain + adjacent problems.
**Option C (Premium)**: Option B + strategic extras (priority support, custom integrations, expanded coverage).

Pricing should be:
- Anchored to the value bridge (not cost-plus)
- Structured so Option B is the obvious choice
- Include a time-based incentive if compelling event exists

### Step 4: Anticipate Objections
Pre-build responses for:
- "Too expensive" -> value bridge, ramp pricing, phased rollout
- "We can build this" -> TCO analysis, time-to-value, maintenance burden
- "Need to think about it" -> compelling event reminder, mutual action plan
- "Competitor is cheaper" -> differentiation on [specific capability]

### Step 5: Close Plan
1. Send agreement principles (not a formal proposal) 48h before pricing call
2. Walk through value bridge live, not over email
3. Get verbal on pricing call, written within 48h
4. Map paper process with specific dates and owners

## Output Format
```
COMMERCIAL PLAN: [Company] | [ACV Target]

VALUE BRIDGE
[Pain] -> [Current Cost] -> [With Us] -> [Delta]
Total 3-Year Value: $[X]

OPTIONS
A. [Scope] - $[price]
B. [Scope] - $[price] (RECOMMENDED)
C. [Scope] - $[price]

OBJECTION PREP
- [Objection]: [Response]

CLOSE TIMELINE
[Date]: Send agreement principles
[Date]: Pricing call
[Date]: Verbal
[Date]: Paper process begins
[Date]: Signed
```

## Never Do
- Never send pricing before quantifying value
- Never discount without getting something in return (timeline, reference, expanded scope)
- Never position against a competitor by name in written materials
- Never fabricate ROI numbers
